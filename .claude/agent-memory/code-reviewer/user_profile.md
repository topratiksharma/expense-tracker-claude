The user (Pratik Sharma, prsa@foss.dk) is the author of a Claude Code course on codewithmosh.com. This expense tracker repo is the course starter project — intentional bugs and rough UI are teaching vehicles, not oversights.

The user is technically proficient enough to set up ESLint flat config, integrate ECharts, build a CSS variable dark/light theme system from scratch, and design a custom delete modal. They understand React hooks and component patterns.

**How to apply:**
- Do not over-explain basic React concepts; communicate at a senior-developer level.
- Flag intentional known bugs but note they may be preserved on purpose for course exercises.
- Accessibility and cross-theme correctness (light/dark) are likely important since this is a course demo students will observe closely.
